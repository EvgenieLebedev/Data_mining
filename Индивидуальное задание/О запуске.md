Запуск докера

    docker build -t myapp:latest -f Run_doc.docker .

Заходим внутрь

    docker run -it myapp:latest /bin/bash

Запускаем внутри докера

    python3 Run_Me.py

Дашборд будет доступен по адресу (загрузка и расчет данных может занять до 10 секунд)

    http://127.0.0.1:8050/
